فكرة موقع إسلامي: "إسلامي جو" (IslamiGo)
موقع إسلامي متكامل يهدف إلى تقديم المحتوى الإسلامي بطريقة حديثة وسهلة الاستخدام، مع ميزات تفاعلية تُساعد المسلمين في حياتهم اليومية.

🛠️ الميزات الرئيسية للموقع:
1. أوقات الصلاة واتجاه القبلة 📍
عرض أوقات الصلاة تلقائيًا حسب موقع المستخدم.
إمكانية اختيار المدينة يدويًا.
عرض عدّ تنازلي للصلاة القادمة.
تحديد اتجاه القبلة باستخدام الـ GPS.
2. القرآن الكريم 📖
عرض القرآن الكريم كاملًا مع خيارات للتكبير والتصغير.
إمكانية البحث عن آية أو سورة.
تفسير الآيات (تفسير ابن كثير، السعدي، الطبري...).
إمكانية تشغيل التلاوة بأصوات قراء مختلفين.
خيار حفظ آخر موضع توقف فيه المستخدم.
3. الأذكار اليومية 🕌
عرض أذكار الصباح والمساء وأذكار بعد الصلاة.
إمكانية التذكير بالأذكار عبر إشعارات.
خيار مشاركة الأذكار مع الآخرين.
4. الفتاوى والاستشارات الشرعية 📜
قسم للفتاوى الشرعية (مستمدة من مصادر معتمدة).
إمكانية إرسال الأسئلة واستقبال الإجابات من علماء معتمدين.
تصنيف الفتاوى حسب المواضيع (الصلاة، الزكاة، الصيام...).
5. مكتبة إسلامية 📚
مقالات ودروس حول العقيدة، الفقه، التفسير، الحديث، والسيرة.
كتب إسلامية مجانية للتحميل.
فهرس لمقاطع فيديو ومحاضرات دينية.
6. التقويم الهجري والمناسبات الإسلامية 📅
عرض التقويم الهجري والميلادي مع إمكانية التبديل بينهما.
تذكير بالمناسبات الإسلامية المهمة (رمضان، العيد، عاشوراء...).
7. خاصية التبرعات والزكاة 💰
إمكانية حساب الزكاة تلقائيًا حسب نوع المال والمبلغ.
دعم التبرعات للجمعيات الخيرية الإسلامية الموثوقة.
إشعار بالتبرعات الشهرية المتكررة.
8. مجتمع المسلمين 🗣️
منتدى إسلامي للنقاشات والأسئلة.
قسم لقصص التوبة والتجارب الشخصية مع الإسلام.
إمكانية إنشاء حساب ومتابعة المستخدمين الآخرين.
9. وضع ليلي / فاتح 🌙☀️
دعم الوضع الليلي لتسهيل القراءة في الليل.
إمكانية التبديل بين الأوضاع تلقائيًا حسب وقت الصلاة.
🎨 التقنيات والتصميم المقترح:
الفرونت إند: Tailwind CSS + React
الباك إند: Node.js + Express.js
قاعدة البيانات: SQLite أو MongoDB (حسب الحاجة)
الخرائط وتحديد المواقع: API من Google Maps لتحديد القبلة وأوقات الصلاة
استضافة: Vercel أو DigitalOcean (حسب الأداء المطلوب)
🎯 الأهداف من الموقع:
✅ تسهيل الوصول إلى المحتوى الإسلامي.
✅ دعم المستخدمين بأدوات دينية مفيدة.
✅ توفير تجربة حديثة ومتقدمة في المحتوى الإسلامي.
✅ المساهمة في نشر العلم الشرعي الصحيح.
